二、基础技术指标介绍

动量指标RSI — 相对强弱指数 (Relative Strength Index)

发明者: J. Welles Wilder (1978年) ｜ 常用周期: 14天

计算方法

第1步: 计算每日价格变化 ΔP = Close(t) - Close(t-1)
第2步: 分离涨幅 Gain = max(ΔP, 0)，跌幅 Loss = max(-ΔP, 0)
第3步: 计算n日平均涨幅和平均跌幅（Wilder平滑法）
AvgGain(t) = [AvgGain(t-1) × (n-1) + Gain(t)] / n
AvgLoss(t) = [AvgLoss(t-1) × (n-1) + Loss(t)] / n
第4步: RS = AvgGain / AvgLoss
第5步: RSI = 100 - 100 / (1 + RS)

作用

  • 超买信号RSI > 70 时，市场可能过度买入，短期有回调风险
  • 超卖信号RSI < 30 时，市场可能过度卖出，短期有反弹机会
  • 背离信号: 股价创新高/新低但RSI未同步，预示趋势可能反转
  • 趋势确认: RSI > 50 表明多头占优，RSI < 50 表明空头占优
RSI取值范围 0~100。在强趋势市场中可能长期处于超买/超卖区域，需结合其他指标使用。
趋势指标MACD — 指数平滑异同移动平均线

发明者: Gerald Appel (1979年) ｜ 常用参数: 12, 26, 9

计算方法

第1步: EMA_fast = 收盘价的12日指数移动平均
EMA(12) = Close(t) × 2/13 + EMA(12, t-1) × 11/13
第2步: EMA_slow = 收盘价的26日指数移动平均
EMA(26) = Close(t) × 2/27 + EMA(26, t-1) × 25/27
第3步: DIF（差离值）= EMA(12) - EMA(26)
第4步: DEA（讯号线）= DIF的9日指数移动平均
第5步: MACD柱 = 2 × (DIF - DEA)

作用

  • 金叉买入DIF 从下向上穿过 DEA，买入信号
  • 死叉卖出DIF 从上向下穿过 DEA，卖出信号
  • 零轴判断: DIF > 0 多头市场，DIF < 0 空头市场
  • 柱状图: 红柱放大多头增强，绿柱放大空头增强
  • 背离: 顶背离（价创新高MACD不创新高）预示见顶；底背离反之
MACD适合趋势行情，在震荡市中可能频繁发出假信号。柱状图变化比交叉信号更提前。
波动率指标布林带 — Bollinger Bands (BOLL)

发明者: John Bollinger (1980年代) ｜ 常用参数: 20日, 2倍标准差

计算方法

中轨(MA) = 过去20日收盘价的简单移动平均
标准差(SD) = 过去20日收盘价的标准差
上轨(Upper) = MA + 2 × SD
下轨(Lower) = MA - 2 × SD
带宽(Bandwidth) = (上轨 - 下轨) / 中轨 × 100%

作用

  • 触上轨价格触及上轨可能超买，有回归均值的需求
  • 触下轨价格触及下轨可能超卖，有反弹的机会
  • 带宽收口: 波动性降低，可能即将出现突破行情
  • 带宽开口: 波动性增大，趋势正在加速发展
  • 中轨支撑/压力: 价格在中轨上方运行为多头趋势
布林带基于统计学正态分布假设，价格约95%时间在上下轨之间波动。突破轨道不一定是反转，可能是趋势启动。

四、扩展指标 — KDJ 随机指标

其他典型技术指标概览

  • KDJ（随机指标） — 基于最高价/最低价/收盘价判断超买超卖，反应灵敏
  • OBV（能量潮） — 用成交量累积衡量资金流向，确认趋势
  • CCI（顺势指标） — 衡量价格偏离均值的程度，识别极端行情
  • ATR（真实波幅） — 衡量市场波动率，常用于止损设置
  • WR（威廉指标） — 类似KDJ的超买超卖振荡器
  • DMI（趋向指标） — 判断趋势强度和方向
  • SAR（抛物线指标） — 动态止损/反转点判断
  • EXPMA（指数平均数） — 比MA更灵敏的趋势线
动量振荡KDJ — 随机指标 (Stochastic Oscillator)

选取 KDJ 作为扩展指标进行介绍和计算。KDJ 是中国市场最常用的短线技术指标之一。

计算方法

第1步: RSV（未成熟随机值）
RSV = (Close - LowestLow(n)) / (HighestHigh(n) - LowestLow(n)) × 100
其中 n 通常为9日，LowestLow为n日最低价，HighestHigh为n日最高价

第2步: K值（RSV的3日加权移动平均）
K(t) = 2/3 × K(t-1) + 1/3 × RSV(t)

第3步: D值（K值的3日加权移动平均）
D(t) = 2/3 × D(t-1) + 1/3 × K(t)

第4步: J值（K和D的偏离度）
J = 3 × K - 2 × D

作用

  • 超买K/D/J > 80 时超买，短期可能回调
  • 超卖K/D/J < 20 时超卖，短期可能反弹
  • 金叉买入: K线从下向上穿过D线（在20以下区域更为可靠）
  • 死叉卖出: K线从上向下穿过D线（在80以上区域更为可靠）
  • J值极值: J > 100 极度超买，J < 0 极度超卖，反转概率大
  • 背离: 与RSI类似，价格与KDJ走势背离预示趋势转折
KDJ 相比 RSI 反应更快、信号更频繁，适合短线操作。但缺点是容易在震荡市中出现假信号。 J 值可以超出 0-100 的范围（理论范围 -100~200），这是 KDJ 区别于 RSI 的一个特点。
