海龟交易策略 · 核心原理
策略概述
海龟交易策略诞生于1983年，由理查德·丹尼斯和威廉·埃克哈特设计。他们招募了一批被称为"海龟"的交易员，用一套完全机械化的趋势跟踪系统进行交易，在4年半内赚取了超过1.5亿美元的利润。

核心思想：趋势跟踪 + 突破入场 + ATR动态风控 + 严格止损。策略不预测市场方向，而是在趋势确认后跟进，通过分散持仓和严格止损控制风险，以少数大盈利交易覆盖多数小亏损交易，实现长期正期望收益。
📊
唐奇安高低点通道
过去N日的最高价和最低价构成通道上下轨。突破上轨买入，跌破下轨卖出。海龟策略使用两套通道：20日入场通道和10日离场通道。
上轨 = max(High, N)
下轨 = min(Low, N)
📡
平均真实波幅 ATR
衡量市场波动性的指标。TR取当日振幅、跳空高开、跳空低开三者最大值，ATR为TR的N日均值。用于动态止损和仓位管理。
TR = max(H-L, |H-Cp|, |L-Cp|)
ATR = SMA(TR, N)
🛡️
止损条件
止损价 = 买入价 - 2×ATR。ATR越大，止损越宽松；ATR越小，止损越紧。仓位管理确保每笔交易风险不超过总资金的2%。
止损价 = Entry - 2 × ATR
仓位 = (资金×2%) / (ATR × 倍数)
关键优势
01

趋势跟踪，捕捉大行情

突破入场确保只在趋势明确时进场，能完整捕捉大级别上涨行情，不错过任何重大机会。

02

ATR自适应波动率

止损距离随市场波动自动调整，高波动时给予更多空间，低波动时收紧止损，智能化程度高。

03

严格的风险管理

每笔交易最大风险控制在2%以内，即使连续亏损也不会伤及本金根基，生存能力极强。

04

规则明确，易于执行

完全机械化的交易规则，消除人为情绪干扰。入场、离场、止损全部由系统信号驱动。

05

多市场适用性

策略逻辑不依赖特定市场结构，可应用于股票、期货、外汇等多个品种，具有广泛适用性。

06

低胜率高盈亏比

典型胜率仅30-40%，但单笔盈利远大于单笔亏损（盈亏比>2:1），长期仍可实现稳定盈利。

2. 唐奇安高低点通道
def calc_donchian_channel(df, entry_period=20, exit_period=10):
    # 入场通道（20日）：突破上轨买入，突破下轨卖出
    df["upper_entry"] = df["high"].rolling(window=entry_period).max().shift(1)
    df["lower_entry"] = df["low"].rolling(window=entry_period).min().shift(1)
    # 离场通道（10日）：跌破10日最低价 → 多头平仓
    df["upper_exit"] = df["high"].rolling(window=exit_period).max().shift(1)
    df["lower_exit"] = df["low"].rolling(window=exit_period).min().shift(1)
    return df
3. ATR (平均真实波幅)
def calc_atr(df, period=20):
    prev_close = df["close"].shift(1)
    tr1 = df["high"] - df["low"]
    tr2 = (df["high"] - prev_close).abs()
    tr3 = (df["low"] - prev_close).abs()
    df["TR"] = pd.concat([tr1, tr2, tr3], axis=1).max(axis=1)
    df["ATR"] = df["TR"].rolling(window=period).mean()
    return df
4. 买卖信号生成
def generate_signals(df, entry_period=20, exit_period=10, stop_atr_mult=2.0):
    # 买入：收盘价 > 入场通道上轨（突破20日新高）
    # 卖出：收盘价 < 离场通道下轨 或 收盘价 < 止损价

使用心得与适应场景
核心发现
通过对9只A股（含3只太空概念股）的回测，我们发现了海龟策略在不同市场环境下的表现差异：

高波动成长股（宁德时代 +29.89%、中国卫星 +158%）—— 趋势跟踪效果最佳
暴跌行情（比亚迪 策略+4.28% vs 买入持有-64.41%）—— 防御力极强
震荡市（贵州茅台、中芯国际）—— 频繁假突破导致连续小亏
稳健蓝筹（三一重工 Sharpe 1.07、MDD仅-11.14%）—— 风控表现最优
适应场景
📈 强趋势市场
当市场处于明确的上升或下降趋势时，海龟策略能完整捕捉大行情。如2024-2025年的航天概念股行情，策略回报达109%-158%。
💧 高波动品种
ATR自适应机制在高波动品种上表现优异。波动越大，趋势越明显，止损越宽，策略越能发挥优势。
⚠️ 震荡市慎用
在横盘震荡行情中，通道突破频繁失效，会产生大量假突破信号，导致连续小亏损。建议结合ADX等趋势强度指标过滤。
⏰ 中长期持有
海龟策略是中长期策略，单笔交易通常持续数周到数月。短线交易者不适合，需要耐心等待突破信号和趋势发展。
使用心得
01 参数选择
经典20/10参数适合大多数品种。A股高波动环境下，紧止损(1.5×ATR)效果更好；低波动品种可用宽止损(3.0×ATR)减少假突破。
02 仓位管理是核心
海龟法则的精髓不在信号而在风控。每笔交易风险控制在总资金的2%以内，即使连续5次亏损也只损失10%本金。
03 多品种分散
单一品种的胜率可能很低，但同时跟踪5-10个不相关品种，只要有1-2个出现大趋势就能覆盖所有亏损并盈利。
04 不要优化过度
过度拟合历史数据会导致未来表现恶化。保持参数简洁（20/10/2.0），不要为了追求高回测收益而频繁调参。
05 接受亏损
海龟策略胜率通常仅30-50%，意味着超过一半的交易会亏损。心理上必须接受"小亏是成本"的理念，严格执行止损。
06 趋势过滤
可加入均线多头排列、ADX>25等趋势过滤条件，减少震荡市的假突破信号。但过滤太多会错过趋势启动的最早机会。
07 加仓规则
原版海龟法则在每上涨0.5×ATR时加仓一个单位，最多4个单位。加仓能放大趋势利润，但也增加了风险，需谨慎使用。
总结
海龟交易策略是一套完整的交易系统，而非简单的技术指标。它的价值在于：① 提供明确的入场/离场/止损规则，消除情绪干扰；② 通过ATR自适应不同品种的波动特性；③ 以严格的风控确保长期生存能力。

在A股市场，海龟策略最适合用于高波动成长股和强趋势行业。通过多品种分散和严格止损，即使在震荡市中也能控制亏损幅度。策略不是万能的，但其系统化的交易思维值得每位量化交易者学习和借鉴。
